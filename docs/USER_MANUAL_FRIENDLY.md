The app window has 4 main areas that work together like a team:

## Common Questions & Quick Fixes

### "The app says my Excel file is invalid"
**Fix:** Check that all 19 column headers are exactly right. Copy and paste from our list above to be sure!

### "My PDF won't open"
**Possible reasons:**
- File is password-protected (remove the password first)
- File is corrupted (try opening it in a PDF reader first)
- File isn't actually a PDF (check the file extension)

### "My formatting disappeared in Excel"
**Fix:** This sometimes happens with solid colored text. Add some **bold** or *italic* to the colored text - that usually fixes it!

### "I can't see all the fields on my screen"
**Fix:** The whole window scrolls! Use your mouse wheel or the scroll bar on the right side.

### "The app is running slowly"
**Solutions:**
- Close other programs to free up memory
- Use smaller PDF files when possible
- Restart the app occasionally during long sessions
